동적 컴포넌트 : 화면의 동일한 위치에 여러 컴포넌트를 표현하기 위해 제공
  component요소를 템플릿에 작성하고 v-bind를 통해 is 특성값으로 위치 지정

  만약 component로 표현할 자식 컴포넌트가 정적 콘텐츠일 경우 keep-alive요소로 감싸주면 매번실행하지 않음
  특정 컴포넌트만 캐싱(매번실행), 언캐싱 원한다면 include, exclulde 특성으로 , 로 구분하여 나열
  (app.vue-about,contact,home.vue)

재귀 컴포넌트 : 템플릿에서 자기 자신을 호출하는 컴포넌트
  name 옵션을 지정해야만 가능
  (recursive, about.vue)
